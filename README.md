# 🔮 The Mind Reader — I Know Your Number

A magical console-based Python game that reads your mind!

## 🎯 What It Does

Think of any number between 1 and 31.
Answer YES or NO for 5 magic cards.
The program reveals your number — every single time.

## ✨ How The Magic Works

It's not actually magic — it's **binary math**!
Each card contains numbers that share a specific binary bit.
Your number = sum of the first number on each card you say YES to.
What feels like mind reading is pure Python logic. 🧠

## 🚀 How To Run

Make sure Python 3 is installed, then:

```bash
python3 mind_reader.py
```

## 🛠️ Concepts Used

- Functions
- Lists & Loops
- Conditionals
- `random` module
- `time` module (dramatic effects!)
- Binary number system logic
